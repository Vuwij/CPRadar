Detailed Description

Interface to the Xethru X4M300 Application module This interface is used to control the XeThru X4M300 module.

See the data sheet of the X4M300 module for more information on the X4M300 module.

§ set_sensor_mode()

int XeThru::X4M300::set_sensor_mode ( const uint8_t  mode,
const uint8_t  param 
)

Control the execution mode of the sensor.

Parameters
[in]modesee xtid.h for profileid values.
Run - 0x01: Start profile execution
Idle - 0x11: Halts profile execution. Can be resumed by setting mode to Run.
Stop - 0x13: Stops profile execution. Must do load_profile to continue.
Manual - 0x12: Routes X4 radar data directly to host rather than to profile execution. Can then interact directly with XEP / X4Driver. Will disrupt profile performance.
[in]paramNot used, ignored, can be 0.
Returns
status : success in case of 0 / failure in any other case
